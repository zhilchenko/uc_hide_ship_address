-- SUMMARY --

Hides delivery address pane from checkout page when specific quote method is selected.
This can be useful for store pickup.

At this moment module is not compatible with uc_addresses



To submit bug reports and feature suggestions, or to track changes:
  zhilchenko@gmail.com



-- REQUIREMENTS --

- Simple Shipping Quotes, tested with v1.0



-- CONFIGURATION --

Check methods which selection will cause hiding delivery address pane on
admin/store/settings/quotes/methods/general.
